# 📡 API Reference

## 🔑 Authentication
`POST /api/auth`
- Handles login, register, and session management.

## 🖼️ Replicate API
`POST /api/replicate`
- Calls AI models for image/video editing.

## 💎 Premium API
`POST /api/premium`
- Manages subscription upgrades and checks.

---

⚠️ All requests require a valid **JWT token** unless otherwise specified.
